# Context Rules

Ignore the following folders to reduce noise and save time when reading the codebase, unless the task specifically requires looking into them (e.g. debugging build issues):

- `**/bin/**`
- `**/obj/**`
- `**/.obj/**`
- `**/node_modules/**`
- `**/.git/**`
- `**/vendor/**`
- `**/*.exe`
- `**/*.dll`
